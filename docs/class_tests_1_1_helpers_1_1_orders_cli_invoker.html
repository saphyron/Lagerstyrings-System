Detailed Description

Helper class to invoke the Orders CLI for testing purposes.

This class provides methods to create orders via the CLI and retrieve results.

async Task< long > Tests.Helpers.OrdersCliInvoker.CreateOrderAsync ( string jsonPayload)

Creates an order by invoking the Orders CLI with the provided JSON payload.

Parameters
jsonPayloadThe JSON payload representing the order.
Returns
A task that represents the asynchronous operation. The task result contains the ID of the created order.

This method writes the JSON payload to a temporary file and invokes the CLI with the file path. It captures the CLI output to extract the order ID.
